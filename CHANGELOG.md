# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)（主版本.次版本.修订号）。
应用内也可在 **首页 ⋯ → 更新日志** 查看。

## v1.1.0 — 2026-06-21

### 新增
- 条目可在编辑模式下按住**右侧 ⠿ 手柄上下拖动排序**。
- 条目可**直接拖动到其他分类**，自动归入该分类（同一区块内：携带物品 ↔ 携带物品、待办 ↔ 待办，不跨区块）。
- 拖动时**其他条目平滑滑动让位**（FLIP 动画），目标位置留出等高空隙，落点一目了然；拿起即在原位留空，列表不跳动。
- 拖动到屏幕上/下边缘时**自动滚动**，便于跨较远分类移动。
- 首页 ⋯ 菜单新增**更新日志**入口，显示当前版本号。

### 说明
- 拖拽用 Pointer Events 实现，兼容 iPhone 触屏（iOS 15+）与桌面鼠标。
- Service Worker 缓存版本升级到 `chuxing-v2`，已安装用户联网打开一次即自动更新。

## v1.0.0 — 2026-06-21

- 首个版本。
- 多份命名清单，按最近使用排序。
- 出差 / 旅游内置模板、空白清单、复制上次清单（勾选自动重置）。
- 携带物品 / 待办事项两大区块，自定义分类与条目。
- 勾选进度统计、实时进度条。
- PWA 离线可用，可添加到 iPhone / 安卓主屏幕。
- 清单备份导出 / 导入（防数据丢失、跨设备迁移）。

# 🧳 外出待办管理

出差、旅游前的 **物品清单 + 待办事项** 工具。装到手机主屏幕，像普通 App 一样用——**完全离线、无需注册、不收集任何数据**。

每次出行存成一份清单，下次出门「复制上次」改两笔就能用,告别每次从零想"我又忘带啥了"。

> **👉 打开使用:** https://godenxugh.github.io/trip-checklist/
> （用手机浏览器打开即可，建议按下方说明装到主屏幕）

---

## ✨ 特点

- 📋 **多份清单**：上海出差、三亚旅游……各存各的，按最近使用排序
- 🧳 **两大区块**：携带物品 / 待办事项，每块下可自定义分类（证件、电子、洗漱…）
- ✅ **打勾即记**：实时进度条，出门前一眼看完没完
- 🆕 **快速复用**：新建时可「复制上次清单」，内容沿用、勾选自动清零
- 🧰 **内置模板**：出差 / 旅游两套常见清单，一键起步再改
- ✎ **随意增删**：加物品、改分类、删条目都行
- 📴 **离线可用**：装好后断网、飞机上照常用
- 🔒 **隐私安全**：数据只存在你自己手机里，不上传任何服务器

## 📱 安装到手机（推荐，体验最好）

### iPhone（用 Safari 打开）
1. 用 **Safari** 打开应用网址（必须 Safari，Chrome 不行）
2. 点底部中间的 **分享按钮**（⬆️ 方框向上箭头）
3. 下滑选 **「添加到主屏幕」** → **添加**
4. 主屏幕出现行李箱图标，点开就是全屏 App

### 安卓（用 Chrome 打开）
1. 用 **Chrome / Edge** 打开应用网址
2. 一般会自动弹 **「安装应用」** 横幅，点它即可
3. 没弹的话：点右上角 **⋮ → 「安装应用 / 添加到主屏幕」**

> 不安装、直接在浏览器里用也完全可以，只是每次要输网址、且离线缓存不如安装后稳定。

---

## 🧭 怎么用

1. **新建清单**：点右下角 **＋**，选出差/旅游模板、空白、或「复制上次清单」
2. **打勾**：点条目即标记完成，再点取消，顶部进度条实时更新
3. **改内容**：每个分类底部输入框可加条目；点右上角 **✎** 进入编辑模式可改名、删条目、加分类
4. **下次复用**：出门前 ＋ → 「复制上次清单」→ 改个名 → 勾选已清零，按需增删即可
5. **清单操作**：在清单内点右上角 **⋯** 可重命名、重置全部勾选、复制、删除

## 💾 关于数据

- 所有清单只保存在**你当前这台手机**上（浏览器本地存储），**不上云、不和别人共享**。
- 换手机或想跨设备：首页 **⋯ → 导出备份**，得到一个 `.json` 文件；在新设备 **⋯ → 导入** 即可恢复。同一份备份在 iPhone、安卓、电脑间通用。
- iOS 对长期不打开的网页 App 可能清理本地数据，建议偶尔导出备份一次以防万一。

## ❓ 常见问题

**要联网吗？** 第一次打开需要联网（下载并缓存），之后日常使用完全离线。

**要注册账号吗？** 不需要，打开即用。

**安卓能用吗？** 能，且安装比 iPhone 更顺手。

**数据会被别人看到吗？** 不会。数据只在你手机本地，别人打开同一网址看到的是空白 App。
